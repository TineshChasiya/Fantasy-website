# Fantasy Cricket Website
A fully responsive fantasy cricket platform only UI. This website contains homepage, join contests, view the preview team, scorecard & completed matches.
# Features
Create login & signup form
Create homepage
Create Captain & Vice-captain page
Create join contests page
Create live & completed matches page
Team previews & stats page
Fully responsive design
Create add money & wallet page
# Tech Stack
Frondend: HTML, CSS, JavaScript 
Tools: VS Code, Git, Github

# Installation
1. Clone the repository
git clone https://github.com/TineshChasiya/Fantasy-website.git
2. Open the project folder
cd fantasy-cricket
Run the first folder /home-page/index.html here is the starting point 
3. Run using Live server/Localhost
